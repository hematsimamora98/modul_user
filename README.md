
# Backend Module User

## How To Run This Server

Install module-user with npm

```bash
  cd server
  npm i
```
Edit your db credentials configuration in ./config/database/config.json

Run the migrations
```bash
  npm run migrate
```
Run role seeder
```bash
  npm run seed
```

Adjust your env variable based on .env.example

Start the server
```bash
  npm run start
```
